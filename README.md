# Semester-Project-Web-Technologies
# Student Learning System by Ahsan Ali(SP24-BSE-004)
# Introduction:
The rapid growth of digital education and online learning platforms has increased the need for efficient systems that facilitate communication and collaboration between students, teachers, and administrators. Traditional methods of managing lectures, assignments, and academic records are often time-consuming, difficult to track, and lack centralized access. To address these challenges, a web-based Student Learning System (SLS) has been developed using the Laravel framework.
The Student Learning System is a role-based educational platform designed to provide separate portals for Admin, Teacher, and Student users. Each user is provided with specific functionalities according to their role. The Admin manages users and system settings, Teachers create courses, upload lectures, manage assignments, and grade submissions, while Students can enroll in courses, access learning materials, submit assignments, view marks, and interact with an AI-powered study assistant.
The system follows the Model–View–Controller (MVC) architecture of Laravel, ensuring a structured, scalable, and maintainable application design. It also incorporates secure authentication and role-based access control to protect resources and restrict access according to user privileges.
In addition, the integration of the Google Gemini API enables an AI Study Assistant that helps students receive instant answers to academic questions. By combining modern web technologies, secure user management, file handling, and artificial intelligence, the Student Learning System provides a comprehensive solution for enhancing the teaching and learning experience in an educational environment.
